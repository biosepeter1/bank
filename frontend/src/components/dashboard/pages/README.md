# Dashboard Pages Structure\n\nThis directory contains all the main page components for the dashboard. Each page is self-contained and manages its own state using Zustand stores.\n\n## Pages Overview\n\n### TransactionsPage.tsx ✅\n- Full transaction history with table view\n- Filter by type, date range\n- Search functionality\n- Export to CSV\n- Status badges\n- Transaction icons (income/expense)\n\n### TransfersPage.tsx (To be created)\n- Local and international transfer forms\n- Beneficiary list and management\n- Transfer history\n- OTP/PIN verification modal\n- Fee calculation display\n\n### VirtualCardsPage.tsx (To be created)\n- Card display with virtual cards\n- Freeze/Unfreeze card buttons\n- Card request form (Visa, MasterCard)\n- Spending limits settings\n- Card usage stats\n\n### DepositsPage.tsx (To be created)\n- Deposit methods selector\n- Payment option forms (Card, Crypto, Bank)\n- Wallet address display\n- Deposit status tracker\n- Upload proof of payment\n\n### CurrencySwapPage.tsx (To be created)\n- Currency pair selector (dropdowns)\n- Real-time conversion rate display\n- Amount input with auto-conversion\n- Swap instantly or schedule\n- Transaction confirmation popup\n- Conversion history\n\n### LoansPage.tsx (To be created)\n- Loan application form\n- Document upload section\n- Loan status tracker (pending, approved, etc.)\n- Active loans display\n- Repayment schedule\n- Loan history\n\n### SettingsPage.tsx (To be created)\n- Personal information edit\n- Security settings (change password, 2FA)\n- Appearance (light/dark mode toggle)\n- Notification preferences\n- Linked accounts\n- Privacy settings\n\n### SupportPage.tsx (To be created)\n- Support ticket form\n- Previous tickets list\n- Live chat widget (placeholder)\n- File attachment for tickets\n- Ticket status tracking\n- FAQ section\n\n## Supporting Components\n\n### overview/ (Dashboard home components)\n- **BalanceCard.tsx** ✅ - Main balance display with visibility toggle\n- **QuickActions.tsx** ✅ - Action buttons for common operations\n- **RecentTransactions.tsx** - Recent transactions preview\n- **SpendingChart.tsx** - Monthly spending visualization\n\n## Styling Standards\n\n- Use Tailwind CSS utilities\n- Dark mode support with `dark:` prefix\n- Icons from `lucide-react`\n- Consistent color scheme:\n  - Blue: Primary actions (buttons, highlights)\n  - Green: Income, success, deposits\n  - Red: Expenses, warnings, withdrawals\n  - Yellow: Alerts, pending items\n\n## State Management\n\nEach page uses Zustand stores for data:\n- `useWalletStore` - Balance and transactions\n- `useTransferStore` - Transfers and beneficiaries\n- `useDepositStore` - Deposits\n- `useWithdrawalStore` - Withdrawals\n- `useCurrencyStore` - Currency rates\n- `useInvestmentStore` - Investments\n- `useLoanStore` - Loans\n- `useSettingsStore` - User preferences\n- `useNotificationStore` - Notifications\n\n## Component Pattern\n\nAll pages follow this structure:\n```tsx\nconst PageName: React.FC = () => {\n  // 1. Get data from stores\n  const { data } = useStore();\n  \n  // 2. Local state for filtering/sorting\n  const [filters, setFilters] = useState({...});\n  \n  // 3. Computed values\n  const filtered = data.filter(...);\n  \n  // 4. Render\n  return (\n    <div>\n      <PageHeader />\n      <Filters />\n      <MainContent />\n    </div>\n  );\n};\n```\n\n## To Complete\n\nCreate remaining page files with similar structure:\n1.
