# Dashboard Components\n\nComprehensive banking dashboard with responsive layout, dark mode support, and modular architecture.\n\n## Directory Structure\n\n```\ndashboard/\n├── layout/                 # Layout components\n│   ├── Sidebar.tsx        # Navigation sidebar\n│   └── DashboardHeader.tsx # Top header with notifications & profile\n├── pages/                 # Page components\n│   ├── overview/          # Overview page sub-components\n│   │   ├── BalanceCard.tsx\n│   │   ├── QuickActions.tsx\n│   │   ├── RecentTransactions.tsx\n│   │   └── SpendingChart.tsx\n│   ├── DashboardOverview.tsx\n│   ├── TransactionsPage.tsx\n│   ├── TransfersPage.tsx\n│   ├── VirtualCardsPage.tsx\n│   ├── RemainingPageStubs.tsx # Deposits, Currency Swap, Loans, Settings, Support\n│   └── index.ts\n├── shared/                # Shared UI components\n│   ├── SharedUIComponents.tsx\n│   └── index.ts\n├── DashboardMain.tsx      # Main router component\n└── index.ts               # Main exports\n```\n\n## Component Usage\n\n### Main Dashboard\n\n```typescript\nimport { DashboardMain } from '@/components/dashboard';\n\n<DashboardMain />\n```\n\n### Pages\n\n```typescript\nimport {\n  DashboardOverview,\n  TransactionsPage,\n  TransfersPage,\n  VirtualCardsPage,\n  DepositsPage,\n  CurrencySwapPage,\n  LoansPage,\n  SettingsPage,\n  SupportPage,\n} from '@/components/dashboard';\n```\n\n### Shared UI Components\n\n#### StatusBadge\nDisplays status indicators (success, pending, warning, error).\n\n```typescript\nimport { StatusBadge } from '@/components/dashboard';\n\n<StatusBadge status=\"success\" label=\"Completed\" size=\"md\" />\n```\n\n#### StatCard\nDisplays statistics with optional trend indicators.\n\n```typescript\nimport { StatCard } from '@/components/dashboard';\nimport { DollarSign } from 'lucide-react';\n\n<StatCard\n  label=\"Total Balance\"\n  value=\"$25,430.00\"\n  change={12.5}\n  trend=\"up\"\n  icon={<DollarSign size={24} />}\n/>\n```\n\n#### TransactionItem\nDisplays individual transaction in a list.\n\n```typescript\nimport { TransactionItem } from '@/components/dashboard';\nimport { ArrowUpRight } from 'lucide-react';\n\n<TransactionItem\n  icon={<ArrowUpRight />}\n  title=\"Money Transfer\"\n  subtitle=\"To John Doe\"\n  amount=\"+$500.00\"\n  date=\"Today, 2:30 PM\"\n  status=\"success\"\n  amountColor=\"positive\"\n/>\n```\n\n#### Form Components\n\nReusable form inputs with validation support.\n\n```typescript\nimport { FormInput, FormSelect, FormTextarea } from **Success**: Green (`green-600`)\n- **Warning**: Yellow (`yellow-600`)\n- **Error**: Red (`red-600`)\n- **Background**: White/Dark gray (`white`/`gray-800`)\n- **Text**: Dark gray/White (`gray-900`/`white`)\n\n### Dark Mode\nAll components support dark mode using Tailwind's `dark:` prefix.\n\n### Responsive\n- Mobile-first design\n- Sidebar collapses on mobile\n- Grid layouts adapt to screen size\n- Touch-friendly buttons and interactions\n\n## State Management\n\nComponents use Zustand stores for state:\n- `userStore` - User profile data\n- `transactionStore` - Transaction history\n- `transferStore` - Transfer history\n- `cardStore` - Virtual card data\n- `depositStore` - Deposit history\n- `currencyStore` - Exchange rates\n- `loanStore` - Loan information\n- `settingsStore` - User preferences\n\n## Icons\n\nUses Lucide React for icons:\n```typescript\nimport { \n  Send, ArrowUpRight, ArrowDownLeft, \n  CreditCard, Settings, Bell, User \n} from 'lucide-react';\n```\n\n## Best Practices\n\n1. **Component Composition**: Break complex components into smaller, reusable pieces\n2. **Dark Mode**: Always test components in both light and dark modes\n3. **Responsive**: Use Tailwind's breakpoints for responsive design\n4. **Accessibility**: Use semantic HTML and ARIA labels\n5. **Performance**: Memoize components and optimize re-renders\n6. **Types**: Use TypeScript interfaces for all props\n7. **Error Handling**: Display user-friendly error messages\n8. **Loading States**: Show loading indicators during async operations\n\n## Future
